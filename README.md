## 插件介绍

> **chalk:** 修改终端颜色  
> **commander:**添加版本命令提示  
> **inquirer:**  命令行交互  
> **minimis":** 命令行解析  
> **ora:**终端旋转器  
> **prettier:** 代码格式化  
> **dayjs:** 时间格式化工具 

## git提交规范

> **feat:** 新功能  
> **fix:** bug 修复
> **docs:** 仅修改文档
> **style:** 修改格式（空格，格式化，省略分号等），对代码运行没有影响
> **refactor:** 重构（既不是修 bug ，也不是加功能）
> **build:** 构建流程、外部依赖变更，比如升级 npm 包、修改 webpack 配置等
> **perf:** 性能优化
> **test:** 测试相关
> **chore:** 对构建过程或辅助工具和库（如文档生成）的更改
> **ci:** ci 相关的更改
